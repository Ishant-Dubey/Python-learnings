# 🧮 Calculator

A simple yet elegant desktop calculator built with Python's Tkinter library. Perform basic arithmetic operations with an intuitive, dark-themed GUI.

---

## ✨ Features

- **Basic Operations**: Addition, Subtraction, Multiplication, and Division
- **Clean Dark Theme**: Modern dark UI with contrasting buttons for better visibility
- **Hover Effects**: Interactive button hover animations for enhanced user experience
- **Error Handling**: Graceful error handling for division by zero
- **Quick Clear**: Clear button (C) to reset the calculator instantly

---

## 📋 Requirements

**For running the Python source code:**
- Python 3.x
- Tkinter (usually included with Python)

**For running the .exe file:**
- No requirements! Just double-click and run (Windows only)

---

## 🚀 Installation & Setup

1. **Clone or download the repository**
   ```bash
   git clone https://github.com/Ishant25Dubey/Python-Learnings.git
   cd Calculator
   ```

2. **Run the calculator**

   **Option A - Using Python (requires Python 3.x installed):**
   ```bash
   python main.py
   ```

   **Option B - Using the standalone .exe file (Windows only):**
   - Simply double-click `Calculator.exe` to run the app directly without needing Python

---

## 🎮 How to Use

1. **Enter Numbers**: Click number buttons (0-9) to input numbers
2. **Select Operation**: Click an operator button (+, −, ×, ÷)
3. **Enter Second Number**: Click number buttons for the second operand
4. **Get Result**: Click the equals (=) button to see the result
5. **Clear**: Click the C button to reset and start a new calculation

### Example Calculation:
- Click `7` → Click `+` → Click `3` → Click `=` → Result: `10`

## 🎨 Customization

You can easily customize the calculator:

- **Colors**: Modify the color codes in the code:
  - `#202020` - Background color
  - `#3B3B3B` - Button color
  - `#323232` - Hover color

- **Font**: Change `'verdana'` to any available font family
- **Window Size**: Modify `root.geometry('272x374')` for different dimensions
- **Button Layout**: Adjust grid positions for custom button arrangement

---

## ⚠️ Limitations

- Only supports basic arithmetic operations
- Cannot chain multiple operations without pressing equals
