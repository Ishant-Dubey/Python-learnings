# 🐍 Python Learnings

A collection of Python projects built while learning and exploring the language. Each project lives in its own folder with its own code and README.

---

## 📁 Projects

| # | Project | Description |
|---|---|---|
| 01 | [QR Code Generator](./QR%20Code%20Generator/) | Generates a QR code from any URL or text and saves it as a PNG |
| 02 | [Wallpaper Viewer](./Wallpaper%20Viewer/) | A Tkinter GUI that loads images from a local folder and lets you cycle through them with a "Next" button |
| 03 | [Calculator](./Calculator/) | A Tkinter desktop calculator with support for basic operations (+, −, ×, ÷) |

> More projects coming soon!

---

## 🛠️ Requirements

Each project has its own dependencies. Check the individual project's README for setup instructions.

General requirements across projects:
- Python 3.x
- pip

---

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Ishant25Dubey/Python-Learnings.git
```

2. Navigate into the project folder you want to run

```bash
cd <project-folder>
```

3. Follow the setup steps in that project's `README.md`
